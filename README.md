# Tanvi Designer Studio

Premium designer kurtis e-commerce website built with Next.js and Tailwind CSS.

## Features

- 🎨 Beautiful, responsive design
- 📦 Product showcase with colors and sizes
- 🛒 Shopping cart functionality
- 📧 Newsletter subscription
- 🔧 Admin dashboard preview
- 📱 Mobile-first approach

## Getting Started

### Prerequisites
- Node.js 14 or higher
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

Deploy on [Vercel](https://vercel.com) - the easiest way to deploy Next.js apps.

## Technologies Used

- Next.js
- React
- Tailwind CSS
